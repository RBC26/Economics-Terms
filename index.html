How to set up scheduled daily reminders

The "Send Now" buttons above generate a personalised email with 8 terms for you to review. For automated daily scheduling (7am & 9pm), you can:

1. Gmail + Zapier: Create a Zap to call this app's email generation and send via Gmail on a schedule
2. Google Calendar: Set a recurring reminder event at 7am and 9pm pointing to this app
3. Manual: Bookmark this tab and tap "Send Now" each morning and evening

Each email contains different terms prioritising ones you haven't learnt yet.
